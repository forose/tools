#vim 에디터 추가기능 설정
<br>
<p>터미널에서 <code>vim ~/.vimrc</code> 입력후에 설정</p>
<p>폰트변경은</p>
<ul>
<li>Mac Os <code>vim ~/.gvimrc</code>
<code><pre>
set macligatures
set guifont=Fira\ Code:h12
set encoding=utf-8
</pre></code>
</li>
<li>Window <code>vim ~/.vimrc</code>
<code><pre>
set guifont=Fira\ Code:h12
set encoding=utf-8
</pre></code>
</li>
